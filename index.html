⚛️ MDAgent2

用于分子动力学代码生成与知识问答的大语言模型

Large Language Model for Code Generation and Knowledge Q&A in Molecular Dynamics

阅读论文 下载 PDF

概述

分子动力学（MD）仿真对于理解材料科学中的原子尺度行为至关重要， 然而编写LAMMPS脚本仍然是一项高度专业化和耗时的任务。 尽管大语言模型在代码生成和领域特定问答方面显示出潜力， 但它们在MD场景中的性能受到领域数据稀缺、最先进LLM的高部署成本以及低代码可执行性的限制。

基于我们之前的MDAgent，我们提出了 MDAgent2， 这是第一个能够在MD领域同时执行知识问答和代码生成的端到端框架。 MDAgent2构建了一个领域特定的数据构建管道，产生了三个高质量数据集， 涵盖MD知识、问答和代码生成。基于这些数据集，我们采用三阶段后训练策略—— 继续预训练（CPT）、监督微调（SFT）和强化学习（RL）—— 训练了两个领域适应模型：MD-Instruct和MD-Code。

主要贡献

  • 提出 MDAgent2，第一个在MD领域同时支持知识问答和代码生成的端到端框架
  • 构建领域特定的数据构建管道，产生三个高质量数据集
  • 采用三阶段后训练策略（CPT、SFT、RL）训练领域适应模型
  • 引入 MD-GRPO，一种利用仿真反馈的闭环强化学习方法
  • 构建 MDAgent2-RUNTIME，可部署的多智能体系统
  • 提出 MD-EvalBench，第一个用于LAMMPS代码生成和问答的基准测试

方法论

三阶段训练策略

MDAgent2采用全面的三阶段后训练策略，实现从通用大语言模型到MD领域专用模型的转换：

  • 继续预训练（CPT）：在MD特定语料库上继续预训练，实现领域适应
  • 监督微调（SFT）：在高质量的指令跟随和代码生成数据集上进行微调
  • 强化学习（RL）：使用MD-GRPO方法通过仿真反馈优化代码生成质量

通过这种分阶段的训练策略，我们训练了两个专门的模型： MD-Instruct（用于知识问答）和MD-Code（用于代码生成）。

MD-GRPO：闭环强化学习

MD-GRPO 是一种新颖的闭环强化学习方法，专门设计用于优化LAMMPS代码生成：

  • 仿真反馈奖励：利用仿真结果作为奖励信号，评估生成代码的质量
  • 轨迹回收机制：回收低奖励轨迹进行持续优化，提高训练效率
  • 迭代改进：通过闭环反馈实现生成代码的迭代改进

MD-GRPO方法能够有效利用仿真环境的反馈信号， 显著提高生成代码的可执行性和正确性。

MDAgent2-RUNTIME：多智能体系统

MDAgent2-RUNTIME 是一个可部署的多智能体系统，实现了完整的代码生成到执行的闭环：

  • 代码生成智能体：从自然语言描述生成LAMMPS脚本
  • 执行智能体：在仿真环境中运行生成的代码
  • 评估智能体：评估代码正确性和仿真结果
  • 纠正智能体：基于执行反馈进行自我纠正和优化

多智能体系统通过协作和分工，实现了从需求理解到代码执行的全流程自动化。

性能评估

我们在MD-EvalBench上的实验结果表明， MDAgent2在代码生成和问答任务上持续超越多个强基线方法， 展示了大语言模型在工业仿真任务中的适应性和泛化能力。

MD-EvalBench

我们建立了 MD-EvalBench，这是第一个用于LAMMPS代码生成和问答的基准测试， 系统评估MDAgent2在以下任务上的能力：

  • 代码生成任务：从自然语言描述生成可执行的LAMMPS脚本
  • 问答任务：回答关于分子动力学的领域特定问题
  • 代码可执行性：确保生成的代码能够在仿真环境中成功运行

实验结果表明，MDAgent2在多个评估指标上持续超越代表性的基线方法， 证明了其在工业仿真任务中的实用价值。
